# Portfolio Website

A modern, responsive portfolio website built with Next.js, React, and TailwindCSS, showcasing professional experience, skills, and projects with stunning animations and interactive components.

## 🚀 Features

- **Modern Tech Stack**: Built with Next.js 16, React 19, and TailwindCSS 4
- **Responsive Design**: Fully responsive layout that works seamlessly across all devices
- **Interactive UI Components**: Powered by Radix UI and custom components
- **Smooth Animations**: Enhanced user experience with smooth transitions and animations
- **AI & Robotics Showcase**: Dedicated section highlighting AI and robotics projects
- **Dark Mode Support**: Built-in theme switching capabilities with next-themes
- **Performance Optimized**: Fast loading times and optimized bundle size
- **Analytics Ready**: Integrated with Vercel Analytics for insights

## 🛠️ Tech Stack

### Frontend
- **Framework**: Next.js 16.0.10
- **UI Library**: React 19.2.0
- **Styling**: TailwindCSS 4.1.9
- **TypeScript**: Full type safety

### UI Components
- Radix UI for accessible, customizable components
- Lucide React for beautiful icons
- Recharts for data visualization
- Embla Carousel for smooth carousels

### Additional Features
- React Hook Form with Zod validation
- Next Themes for dark mode
- Sonner for toast notifications
- Class Variance Authority for component variants

## 📦 Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Sujith-2210/Sujith.git
   cd Sujith
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   pnpm install
   # or
   yarn install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   # or
   pnpm dev
   # or
   yarn dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the result.

## 🚀 Deployment

### Deploy on Vercel

The easiest way to deploy your Next.js app is to use the [Vercel Platform](https://vercel.com/new).

1. Push your code to GitHub
2. Import your repository on Vercel
3. Vercel will automatically detect Next.js and configure the build settings
4. Click "Deploy"

Check out the [Next.js deployment documentation](https://nextjs.org/docs/app/building-your-application/deploying) for more details.

### Deploy on Other Platforms

This project can also be deployed on:
- Netlify
- AWS Amplify
- Railway
- Render

Refer to the respective platform's documentation for Next.js deployment.

## 🎨 Customization

### Updating Content
- Modify components in the `components/` directory
- Update pages in the `app/` directory
- Change global styles in `styles/globals.css`

### Theming
- Customize colors and design tokens in TailwindCSS configuration
- Update theme settings with next-themes
